Git常用命令

发表于 2018-08-18 分类于 代码库

Gitee 提供免费的 Git 仓库，还集成了代码质量检测、项目演示等功能。对于团队协作开发，Gitee 还提供了项目管理、代码托管、文档管理的服务，5 人以下小团队免费。

接下来我们学习一下如何使用 Gitee。

Git 的工作流程。

一般工作流程如下：

  • 克隆 Git 资源作为工作目录。
  • 在克隆的资源上添加或修改文件。
  • 如果其他人修改了，你可以更新资源。
  • 在提交前查看修改。
  • 提交修改。
  • 在修改完成后，如果发现错误，可以撤回提交并再次修改并提交。

首先登录Gitee

https://gitee.com/

新建仓库


git常用命令

将存储库的内容克隆到本地

git clone 链接地址

首先,作为 git 的基础配置，作用是告诉 git 你是谁，你输入的信息将出现在你创建的提交中。然后初始化项目

git config - -global user.name “你的名字或昵称”
git config - -global user.email “你的邮箱”
git init

将所有内容添加到暂存区 .代表所有

git add .

存储库记录更改

git commit -m “更改说明”

将仓库和服务器连接

git remote add origin 链接地址

删除远程地址

git remote rm origin

将本地仓库提交到远程仓库

git push origin master

创建分支并切换过去

git checkout -b 分支名

切换回主分支

git checkout master

删除分支

git branch -d 分支名

更新本地仓库

git pull

将分支推送到远程仓库

git push origin 分支名

查看仓库当前的状态，显示有变更的文件。

git status
