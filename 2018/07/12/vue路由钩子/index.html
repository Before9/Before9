VUE路由钩子

Created2018-07-12Updated2021-01-28Visited

描述：通常我们在跳转页面时会遇到各种各样的要求，比如没有登录不能查看，比如跳转需要验证，各种各样的要求。如果分开写代码的话就会显得很乱，代码很繁重，此时就用到了我们的路由钩子
路由钩子主要是给使用者在路由发生变化时进行一些特殊的处理而定义的

路由钩子主要分三种

全局路由钩子：beforeEach
单个路由钩子：beforeEnter
路由组件钩子：beforeEachEnter

三种路由钩子中都涉及到了三个参数:

to: Route: 即将要进入的目标 路由对象
from: Route: 当前导航正要离开的路由
next: Function: 通过这个方法来调用下一个钩子函数

使用：

beforeEach(跳转前) && afterEach(跳转后)
需要通过main.js配置

beforeEnter(写在router-index.js里面)

beforeRouteEnter(到达之后) && beforeRouteUpdate(到达子路由) && beforeRouteLeave(离开之前)
写在具体页面里
