python数据类型常用操作方式

发表于 2018-08-16 分类于 后端

python常用数据类型

  • 数字
  • 字符串
  • 列表
  • 元祖
  • 字典
  • 集合

列表常用方法

append ：用于在列表末尾添加新的元素
extent : 合并两个列表
insert : 将元素插入到列表指定位置
count : 统计某个元素在列表中出现的次数
index : 用于从列表中找出来某个值第一个匹配项的位置
reverse ： 将列表中的元素反向存放
sort ：对列表进行排序

sort和sorted的区别是什么？
sort是用来对列表进行排序的(无返回值)，而sorted可以对所有可迭代对象排序(返回一个新的列表)

pop : 删除列表末尾元素
remove ：移除列表中某个值的第一个匹配项

字符串常用方法

find：找个某个字符串第一次出现的位置
join：拼接字符串(逆方法 split)
split：拆分字符串
strip：去除首尾空格
replace：替换字符串
count：查询某个字符在字符串中出现的次数

字典的常用方法

clear：清空字典
fromkeys：创建一个新字典，序列元素作为键，值为默认值
get：获取指定键的值
setdefault：获取指定键的值，如果键不存在那么添加一个新的键值对
pop：删除指定键的值**
update：更新字典，可以用来合并字典
keys:返回一个包含所有键的列表
values:返回一个包含所有值的列表
items:返回一个包含所有键值对的列表

集合常用方法

add：添加
clear：清空集合
update：合并集合
pop：随机删除一个元素

set：去重
intersection：交集
union：并集
difference：差集

元祖常用方法

cmp(tuple1, tuple2)：比较两个元组元素。
len(tuple)：计算元组元素个数。
max(tuple)：返回元组中元素最大值。
min(tuple)：返回元组中元素最小值。
tuple(seq)：将列表转换为元组。
